# Community Workflows

A collection of ready-to-use workflows for Instructionly, created and maintained by the community.

## What are Workflows?

Workflows are visual process diagrams that document step-by-step procedures. They consist of nodes (steps) connected by edges (transitions), allowing you to map out complex processes like code reviews, deployments, onboarding, and more.

## How to Use

### Option 1: Import via Instructionly App

1. Browse the workflows below and find one you like
2. Click the workflow file to view it on GitHub
3. Click "Raw" to get the raw JSON
4. Copy the JSON content
5. In Instructionly, go to **Workflows** → **Import**
6. Paste or drag-drop the JSON file
7. Done! The workflow is now in your workspace

### Option 2: Download and Import

1. Download the `.json` file
2. In Instructionly, go to **Workflows** → **Import**
3. Drag and drop the file
4. Done!
