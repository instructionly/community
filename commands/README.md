# Community Commands

A collection of ready-to-use slash commands for Instructionly, created and maintained by the community.

## How to Use

### Option 1: Import via Instructionly App

1. Browse the commands below and find one you like
2. Click the command file to view it on GitHub
3. Click "Raw" to get the raw JSON
4. Copy the JSON content
5. In Instructionly, go to **Commands** → **Import**
6. Paste or drag-drop the JSON file
7. Done! The command is now in your workspace

### Option 2: Download and Import

1. Download the `.json` file
2. In Instructionly, go to **Commands** → **Import**
3. Drag and drop the file
4. Done!
